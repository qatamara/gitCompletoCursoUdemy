GITHUB : Um servidor remoto para armazenamento de codigo, é onde armazenamos o nosso codigo.
Comandos basicos para um bom desempanho no terminal:

  • CLEAR-limpa terminal
  • DIR-lista as pastas
  • CD-entra nas pastas
  • MKDIR-cria uma pasta
  • ECHO-printa uma frase no terminal
  • RMDIR-remove um repositorio
